Whisper STT — macOS Setup

System-wide local speech-to-text via whisper.cpp + Hammerspoon. Works in any app: Terminal, Claude Code, browser, Outlook…

Hotkeys

Shortcut Action
Ctrl+Cmd+W Toggle recording — first press starts, second press stops + transcribes + pastes
Ctrl+Cmd+Q Stop recording immediately (emergency stop)

Menubar Indicator

Symbol State
○ Idle
● 0:12 Recording (with elapsed time)
⠋⠙⠹… Transcribing (whisper-cli running)

Dropdown Menu

Clicking the menubar icon shows Toggle / Stop controls, Notifications / Sounds toggles, a model selector to switch between all .bin files in models/, and a list of recent transcriptions. Click any history entry to copy it to the clipboard.

Architecture

Ctrl+Cmd+W
   ↓
Hammerspoon (whisper_hotkeys.lua)
   ↓ hs.task.new()
whisper.sh toggle
   ├─ [start] ffmpeg records → /tmp/whisper_recording.wav
   └─ [stop]
        ├─ touch /tmp/whisper_transcribing  ← spinner startet
        ├─ whisper-cli transcribes
        ├─ rm /tmp/whisper_transcribing  ← spinner stoppt
        ├─ append to history.txt (rolling 10)
        ├─ copy to clipboard
        └─ auto-paste via AppleScript

Variable Default Description

WHISPER_TRANSLATE 0 Set 1 to translate output to English

WHISPER_NOTIFICATIONS 1 Set 0 to disable notifications

Manual Testing

~/Scripts/Whisper/whisper.sh status            # current state

Pitfalls & Common Mistakes

  • Hotkey does nothing — Hammerspoon config not reloaded. Menubar → Reload Config.
  • No audio output — Wrong device index. Run whisper.sh list-devices and adjust WHISPER_AUDIO_DEVICE.
  • whisper-cli not found — PATH not available in Hammerspoon. find_bin() checks Homebrew paths first.
  • Spinner stays on — /tmp/whisper_transcribing not removed. Auto-cleaned after 10 min; check error path in stop_recording().
  • Hotkey ignored during transcription — Expected behaviour. The toggle hotkey is locked while whisper-cli is running. Wait for the transcription to finish.
  • Model rejected for language — .en.bin models refuse non-English. Switch model or set WHISPER_LANGUAGE=en.
  • F5 / Cmd+numbers — macOS Dictation owns F5; many apps use Cmd+number for tabs. Use Ctrl+Cmd combos.

Uninstall

./uninstall.sh

Removes the Hammerspoon integration and temp files. Optionally uninstalls whisper-cpp, ffmpeg and Hammerspoon via Homebrew (asks before each). The repo directory is kept — delete manually if desired.
